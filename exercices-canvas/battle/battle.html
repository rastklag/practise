Un aventurier (rond rouge) perdu dans un labyrinthe…
  • espace = endroit où on peut passer
  • # = mur (bloc gris)
  • | = porte fermée (blocs blancs)
  • . = clé (point bleu)
  • Toutes les clés ouvrent toutes les portes… mais les clés disparaissent après usage (sinon c'est pas drôle).
Ton premier exercice : améliorer le labyrinthe pour que ce soit un minimum intéressant (pour le moment, on n'a même pas besoin d'ouvrir la porte pour gagner). Pour t'aider à "sculpter" le labyrinthe, mets un espace dans le petit input nommé "draw-char" → le joueur pourra alors "creuser" dans les murs. Si tu veux "dessiner" des murs, mets un # à la place. La version texte se mettra à jour dans le textarea (invisible mais que tu peux afficher pour le copier-coller à la fin). Pour dessiner juste 1 mur, appuie sur 'e' puis déplace-toi: la case où tu étais se transforme en mur.
Un X (bloc jaune) marque la sortie du labyrinthe et donc la victoire.
