But de l'exercice : modifier 'script-exercice.js' pour ajouter, dans la div "contenu", un tableau HTML qui présente les gares dont les données sont disponibles dans l'objet `dataGares`.
  • Colonnes: "commune", "nom de la gare", "département", "géolocalisation"
  • Une ligne d'en-tête décrivant les colonnes (utiliser thead et th)
  • Une ligne par gare dans le tbody
  • Colonne 1 = "Commune"
  • Colonne 2 = "Gare" (champ 'libelle')
  • Colonne 3 = "Département"
  • Colonne 4 = "Géolocalisation" (champ 'coordinates' de l'objet 'geometry')
Bonus: mettre une classe sur chaque td. Bonus extra: grâce à cette classe, permettre de choisir les colonnes à afficher.
